# 用户使用手册功能设计

## 概述

为 POS 设备管理系统创建一个完整的用户使用手册页面，帮助用户了解和使用系统各项功能。

## 需求总结

- **内容范围**：完整系统手册，覆盖所有功能模块
- **展示方式**：独立页面，新增"帮助中心"菜单项
- **内容组织**：分类导航式，左侧分类菜单，右侧显示内容
- **搜索功能**：不需要
- **语言**：仅中文
- **图片**：纯文字说明

## 技术方案

采用静态页面 + 硬编码内容方式：
- 创建 `HelpPage.jsx` 组件，帮助内容直接写在组件中
- 无需后端支持，实现简单快速
- 页面加载快，无需额外请求

## 导航集成

在 Navbar 左侧菜单添加"帮助中心"入口，位于"管理中心"之前：

```
POS设备 | 移动设备 | 工作台 | 管理中心 | 帮助中心
```

## 技术实现要点

1. 新建 `HelpPage.jsx` 页面组件
2. 左侧菜单使用固定定位，右侧内容区可滚动
3. 点击左侧菜单项平滑滚动到对应内容
4. 保持 Apple Design System 设计风格
5. 添加路由配置 `/help`
6. 在 Navbar 中添加导航入口

## 角色权限说明

帮助中心页面：所有登录用户均可访问
